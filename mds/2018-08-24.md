## linux安装 anaconda

```
# 下载anaconda
$ wget https://repo.continuum.io/archive/Anaconda3-5.0.1-Linux-x86_64.sh
```
![image_1clliuonc3phcancg6bjoenp.png-192.2kB][1]
```
$ bash Anaconda3-5.0.1-Linux-x86_64.sh
```
```
#添加环境变量
$ echo 'export PATH="~/anaconda3/bin:$PATH"' >> ~/.bashrc
```
